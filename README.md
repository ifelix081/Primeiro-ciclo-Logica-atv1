# Questoes de Logica - C

Menu interativo em C com as 15 questoes de logica de programacao. Cada questao exibe seu enunciado e executa o algoritmo correspondente.

## Questoes disponíveis

| # | Tema |
|---|------|
| 1 | Multiplicacao de dois numeros |
| 2 | Antecessor e sucessor |
| 3 | Quociente e resto da divisao |
| 4 | Soma, produto e quociente |
| 5 | Conversao de centimetros para polegadas |
| 6 | Area do circulo |
| 7 | Area e volume do cilindro |
| 8 | Salario com comissao de 15% |
| 9 | Troca de variaveis |
| 10 | Cedulas do caixa eletronico |
| 11 | Troco do supermercado em cedulas e moedas |
| 12 | Media aritmetica |
| 13 | Preco final do automovel com impostos |
| 14 | Positivo, negativo ou zero |
| 15 | Par ou impar |

## Como usar

Ao rodar o programa, um menu e exibido. Digite o numero da questao desejada e pressione Enter. O enunciado aparece antes de pedir os dados. Ao terminar, pressione Enter para voltar ao menu. Digite `0` para sair.

## Requisitos

- Compilador GCC (ou qualquer compilador C padrao)
- Terminal com suporte a entrada pelo teclado
